# DBA Registration Guide

## How to Register a DBA in Texas Under Denvan Holdings LLC

---

## What Is a DBA?

DBA stands for "Doing Business As" (also called an Assumed Name). It allows Denvan Holdings LLC to operate a business under a different name without forming a new legal entity.

**Example:** Denvan Holdings LLC registers a DBA called "Austin Marketing Pro" to run a marketing services business.

---

## Step-by-Step Process

### Step 1: Choose a Business Name
- [ ] Verify the name isn't already taken: [sos.state.tx.us](https://www.sos.state.tx.us/corp/sosda/index.shtml)
- [ ] Make sure it doesn't infringe on existing trademarks: [uspto.gov](https://www.uspto.gov/trademarks)
- [ ] Name must not include "LLC", "Inc", "Corp" or other entity designations
- [ ] Name should be distinct from other DBAs in the portfolio

### Step 2: File Assumed Name Certificate with County Clerk
- **Where:** Travis County Clerk's Office (for Austin-based businesses)
  - Address: 5501 Airport Blvd, Austin, TX 78751
  - Online filing available at: [countyclerk.traviscountytx.gov](https://countyclerk.traviscountytx.gov)
- **Form:** Assumed Name Certificate (Form 503)
- **Cost:** $25 filing fee
- **Processing:** 1-3 business days
- **Required Information:**
  - Assumed name (DBA name)
  - Entity filing the DBA: "Denvan Holdings LLC"
  - Entity type: Limited Liability Company
  - State of formation: Texas
  - Office address: 5900 Balcones Drive, STE 100, Austin, TX 78731
  - Period of use: 10 years (renewable)
  - Counties where business will be conducted

### Step 3: File with Texas Secretary of State (if operating in multiple counties)
- **When required:** If doing business in more than one county
- **Where:** Texas Secretary of State
- **Form:** Assumed Name Certificate for an Incorporated Business or LLC (Form 503)
- **Cost:** $25 filing fee
- **Note:** File county AND state if operating beyond Travis County

### Step 4: Post-Registration Setup
- [ ] Open bank sub-account: "DH - [DBA Name]"
- [ ] Set up in QuickBooks as a new Class
- [ ] Create Google Drive folder
- [ ] Set up business email ([name]@denvanholdings.com)
- [ ] Create branding assets
- [ ] Draft Management Services Agreement
- [ ] Follow the New Subsidiary Launch Checklist

---

## Renewal

- Texas DBA registrations are valid for **10 years**
- Renewal requires filing a new Assumed Name Certificate before expiration
- Set a calendar reminder for 11 months before expiration
- Same filing fee ($25) for renewal

---

## Abandonment (Closing a DBA)

If you decide to close a subsidiary:
1. File an **Abandonment of Assumed Name** with the county clerk
2. File with Secretary of State (if filed there originally)
3. Close the bank sub-account
4. Update QuickBooks and financial records
5. Remove from website and marketing materials

## How to Use This Manual

1. **Starting from scratch?** Begin with `01-company-setup/` and work through in order
2. **Launching a new subsidiary?** Go to `03-subsidiary-management/new-subsidiary-launch-checklist.md`
3. **Day-to-day operations?** Use `08-daily-operations/daily-weekly-monthly-checklist.md`
4. **Need a template?** Check `05-legal-templates/` or `04-financial-operations/invoice-templates.md`
5. **Planning growth?** See `09-growth-playbook/growth-strategy.md`

All templates use `[BRACKETS]` for fields you need to fill in with your specific information.

## Reporting Structure

### Weekly
```
Subsidiary Manager вҶ’ Parent Company (CEO/COO)
  - Revenue update
  - Key wins / blockers
  - Customer pipeline status
  - 15-minute check-in call
```

### Monthly
```
Subsidiary Manager вҶ’ Finance Team вҶ’ CEO
  - Full P&L per subsidiary
  - KPI dashboard
  - Customer metrics
  - 45-minute performance review
```

### Quarterly
```
All Subsidiary Managers + CEO + Finance
  - Portfolio-wide strategic review
  - Budget vs. actual analysis
  - Growth planning
  - Cross-subsidiary collaboration opportunities
  - 2-hour strategy session
```

### Annually
```
CEO + Advisory Board (if applicable)
  - Annual report and financial summary
  - Strategic plan update
  - Portfolio evaluation
  - Growth targets for next year
  - Half-day planning session
```

## QuickBooks Setup Notes

1. **Classes:** Create a class for each subsidiary (Sub A, Sub B, Sub C, Parent/Corporate)
2. **Assign every transaction** to a class so you can run P&L by subsidiary
3. **Sub-accounts:** Use the numbering above for easy navigation
4. **Add accounts** as needed when launching new subsidiaries (follow the numbering pattern)

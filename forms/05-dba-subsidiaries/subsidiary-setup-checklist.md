## Phase 2: Legal Setup

### DBA Registration
- [ ] File Assumed Name Certificate with County Clerk
  - County: [COUNTY]
  - Filing date: [DATE]
  - Filing number: [NUMBER]
- [ ] File Form 503 with Texas Secretary of State
  - Filing date: [DATE]
  - Filing number: [NUMBER]

### Tax Setup
- [ ] Obtain separate EIN for subsidiary (if needed): [EIN]
  - Note: A DBA can use the parent company's EIN, but a separate EIN may be preferred for banking and accounting separation
- [ ] Register for Texas Sales Tax Permit (if selling taxable goods/services)
  - Permit number: [NUMBER]
- [ ] Determine payroll tax obligations (if subsidiary will have employees)

### Banking
- [ ] Open separate business bank account under DBA
  - Bank: [BANK NAME]
  - Account number: [LAST 4 DIGITS]
- [ ] Set up merchant services / payment processing (if applicable)
- [ ] Set up accounting software / chart of accounts

### Contracts & Agreements
- [ ] Draft subsidiary operating procedures
- [ ] Prepare standard service agreements / contracts
- [ ] Prepare client/customer terms and conditions
- [ ] Review insurance needs and obtain coverage

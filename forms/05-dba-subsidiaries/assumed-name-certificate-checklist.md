# Assumed Name Certificate (DBA) Checklist

## Overview

Texas law requires a business entity to file an Assumed Name Certificate when operating under any name other than its legal entity name. Denvan Holdings LLC must file a DBA for each subsidiary that operates under a different name.

---

## Filing Requirements

### 1. County Filing (Required)

File an Assumed Name Certificate with the County Clerk in **each county where business is conducted** under the assumed name.

| Field | Details |
|-------|---------|
| Where to file | County Clerk's office |
| Fee | Typically $25-50 (varies by county) |
| Form | Assumed Name Certificate (varies by county) |
| Duration | Up to 10 years |
| Renewal | Must renew before expiration |

## Abandonment of Assumed Name

To stop using a DBA, file **Form 504** (Abandonment of Assumed Name) with the Texas SOS.

- Filing fee: $25
- Also notify the County Clerk in each county where the DBA was filed

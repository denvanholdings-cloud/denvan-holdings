# Annual Compliance Calendar

## Denvan Holdings LLC

**Fiscal Year:** January 1 -- December 31
**Tax Year:** [YEAR]

---

## January

- [ ] **Jan 15** -- Q4 estimated tax payment due (Form 1040-ES for members)
- [ ] **Jan 20** -- Annual sales tax return due (if annual filer)
- [ ] **Jan 31** -- Issue W-2s to all employees
- [ ] **Jan 31** -- Issue 1099-NEC to all independent contractors ($600+)
- [ ] **Jan 31** -- File 1099-NEC with IRS
- [ ] **Jan 31** -- File Form 940 (Annual FUTA return)
- [ ] **Jan 31** -- File Form 941 (Q4 quarterly payroll return)
- [ ] Review and update BOI report if any ownership changes occurred

## February

- [ ] **Feb 28** -- File 1099-MISC with IRS (March 31 if e-filing)
- [ ] **Feb 28** -- File W-2/W-3 with Social Security Administration (if paper; Jan 31 if e-filing)
- [ ] Begin gathering financial records for tax return preparation
- [ ] Review prior year revenue to determine franchise tax filing category

## March

- [ ] **Mar 15** -- Partnership (Form 1065) or S-Corp (Form 1120-S) tax return due
  - Or file extension (Form 7004) by this date
- [ ] **Mar 15** -- Distribute Schedule K-1 to all members
- [ ] Prepare franchise tax forms (review revenue, determine category)
- [ ] Review registered agent status with Texas SOS

## April

- [ ] **Apr 15** -- Individual tax returns due (members' personal returns)
- [ ] **Apr 15** -- Q1 estimated tax payment due
- [ ] **Apr 15** -- C-Corp tax return due (Form 1120), if applicable
- [ ] **Apr 30** -- File Form 941 (Q1 quarterly payroll return)
- [ ] Finalize Texas Franchise Tax Report and Public Information Report
- [ ] Review and file early if possible (deadline is May 15)

## May

- [ ] **MAY 15** -- **TEXAS FRANCHISE TAX REPORT DUE** (CRITICAL)
- [ ] **MAY 15** -- **PUBLIC INFORMATION REPORT (Form 05-102) DUE** (CRITICAL)
- [ ] File No Tax Due Report (Form 05-163) if revenue below $2.47M
- [ ] OR file E-Z Computation (Form 05-169) or Long Form (05-158-A/B)
- [ ] Request extension to Nov 15 if needed (must pay 90% of estimated tax)
- [ ] Save filing confirmation number

## June

- [ ] **Jun 15** -- Q2 estimated tax payment due
- [ ] Mid-year financial review
- [ ] Review subsidiary DBA renewal dates
- [ ] Verify all sales tax filings are current

## July

- [ ] **Jul 31** -- File Form 941 (Q2 quarterly payroll return)
- [ ] Mid-year compliance review:
  - [ ] All tax filings current?
  - [ ] Registered agent active?
  - [ ] Insurance policies current?
  - [ ] BOI report accurate?

## August

- [ ] Review operating agreement -- any amendments needed?
- [ ] Evaluate need for new subsidiary launches
- [ ] Update business continuity plans
- [ ] Review employee handbook and policies

## September

- [ ] **Sep 15** -- Q3 estimated tax payment due
- [ ] **Sep 15** -- Extended partnership/S-Corp tax returns due (if extension filed)
- [ ] Begin year-end tax planning with CPA
- [ ] Review capital expenditure plans for Q4

## October

- [ ] **Oct 15** -- Extended individual/C-Corp tax returns due (if extension filed)
- [ ] **Oct 31** -- File Form 941 (Q3 quarterly payroll return)
- [ ] Schedule annual member meeting (Q4)
- [ ] Review compliance audit checklist
- [ ] Verify all DBA certificates are current

## November

- [ ] **Nov 15** -- Extended Texas Franchise Tax Report due (if extension filed)
- [ ] Year-end financial projections
- [ ] Plan distributions and tax optimization strategies
- [ ] Review employee benefits for open enrollment

## December

- [ ] Conduct annual member meeting and document minutes
- [ ] Year-end review of all compliance items
- [ ] Close books for the fiscal year
- [ ] Review and renew business insurance policies
- [ ] Review and update BOI report if any changes pending
- [ ] Prepare for January compliance deadlines
- [ ] Archive current year records per retention schedule

- [ ] Keep corporate records book current

- [ ] File monthly/quarterly payroll returns (if applicable)

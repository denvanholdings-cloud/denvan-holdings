# Denvan Holdings LLC - Subsidiary Integration Outline

## Overview

This document provides a step-by-step framework for integrating new subsidiaries into the Denvan Holdings ecosystem. It covers legal, financial, branding, operational, and technical integration.

---

## Phase 1: Pre-Launch Planning (Weeks 1-2)

### 1.1 Business Validation
- [ ] Market research completed
- [ ] Competitive analysis documented
- [ ] Financial projections prepared (12-month minimum)
- [ ] Business plan drafted
- [ ] Alignment with Denvan Holdings portfolio strategy confirmed

### 1.2 Legal Structure Decision
- [ ] Determine: DBA or Separate LLC
- [ ] If DBA: File "Doing Business As" with county/state
- [ ] If LLC: File Articles of Organization with state
- [ ] Draft Operating Agreement (template from parent)
- [ ] Obtain EIN from IRS (if separate LLC)
- [ ] Register for state/local business licenses as needed

### 1.3 Financial Setup
- [ ] Open subsidiary bank account (same bank as parent preferred)
- [ ] Set up subsidiary in QuickBooks/accounting software
- [ ] Define management fee structure (5-15% of revenue to parent)
- [ ] Create initial budget
- [ ] Set up payment processing (Stripe, Square, etc.)
- [ ] Establish invoicing templates

---

## Phase 2: Branding & Digital Setup (Weeks 2-3)

### 2.1 Branding
- [ ] Choose subsidiary name
- [ ] Apply Denvan Holdings brand framework:
  - Subsidiary logo using parent brand guidelines
  - Assign sector accent color
  - Use Inter font family
  - Add "A Denvan Holdings Company" endorsement
- [ ] Create brand assets:
  - Logo files (SVG, PNG in multiple sizes)
  - Business card template
  - Email signature template
  - Letterhead template
  - Social media profile/cover images

### 2.2 Digital Presence
- [ ] Set up email addresses (subsidiary subdomain or separate domain)
- [ ] Configure DNS records (MX, SPF, DKIM, DMARC)
- [ ] Create subsidiary section on Denvan Holdings website OR set up subdomain
- [ ] Set up social media profiles (LinkedIn, Instagram at minimum)
- [ ] Create Google Business Profile (if applicable)
- [ ] Set up scheduling tool (Calendly)

### 2.3 Technology
- [ ] Add team to Google Workspace (or chosen platform)
- [ ] Set up shared drives for subsidiary documents
- [ ] Create subsidiary channels in team communication tool
- [ ] Add to project management workspace (Notion, Asana)
- [ ] Issue password manager invitations
- [ ] Set up any industry-specific software

---

## Phase 3: Operations Setup (Weeks 3-4)

### 3.1 Service Definition
- [ ] Define service offerings with clear descriptions
- [ ] Set pricing strategy
- [ ] Create service agreements / contracts (use parent templates)
- [ ] Build sales collateral (one-pager, pitch deck)

### 3.2 Standard Operating Procedures
- [ ] Client onboarding process
- [ ] Service delivery workflow
- [ ] Quality assurance checklist
- [ ] Client communication standards
- [ ] Billing and collections process
- [ ] Complaint/issue resolution process

### 3.3 Team
- [ ] Define roles and responsibilities
- [ ] Hire or assign team members
- [ ] Conduct onboarding training
- [ ] Set up payroll (if applicable)
- [ ] Create performance metrics and KPIs

---

## Phase 4: Marketing & Launch (Weeks 4-6)

### 4.1 Pre-Launch Marketing
- [ ] Create launch announcement content
- [ ] Prepare email announcements for existing network
- [ ] Schedule social media launch posts
- [ ] Set up CRM pipeline in HubSpot
- [ ] Create lead capture forms

### 4.2 Soft Launch
- [ ] Launch to limited audience / warm network
- [ ] Gather initial feedback
- [ ] Refine service delivery based on feedback
- [ ] Test all systems end-to-end (booking, billing, delivery)

### 4.3 Full Launch
- [ ] Public announcement across all channels
- [ ] Blog post on Denvan Holdings website
- [ ] Newsletter announcement
- [ ] Begin ongoing marketing campaigns
- [ ] Activate SEO strategy for subsidiary

---

## Phase 5: Ongoing Management

### Monthly
- [ ] Financial review with parent company
- [ ] Performance metrics review (KPIs)
- [ ] Team check-in and feedback session
- [ ] Marketing performance review
- [ ] Client satisfaction assessment

### Quarterly
- [ ] Strategic planning session with parent company leadership
- [ ] Budget review and adjustment
- [ ] Service offering review and updates
- [ ] Team training and development
- [ ] Technology and tools audit

### Annually
- [ ] Comprehensive business review
- [ ] Financial audit
- [ ] Strategic plan update
- [ ] Brand refresh assessment
- [ ] Growth planning for next year

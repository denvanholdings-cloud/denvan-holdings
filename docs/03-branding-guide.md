# Denvan Holdings LLC - Brand Style Guide

## Logo Concepts

### Primary Logo
The Denvan Holdings logo features a bold, geometric "D" monogram integrated with an upward-pointing architectural element, symbolizing growth, structure, and stability. The full company name sits beside or below the monogram in clean, modern typography.

### Logo Variations
1. **Full Logo** - Monogram + "DENVAN HOLDINGS" wordmark (primary use)
2. **Monogram Only** - "D" mark for favicons, social media profiles, watermarks
3. **Wordmark Only** - Text-only version for documents and email signatures
4. **Stacked Logo** - Monogram above wordmark for square formats
5. **Horizontal Logo** - Monogram left of wordmark for headers and banners

### Logo Usage Rules
- Minimum clear space: Equal to the height of the "D" on all sides
- Minimum size: 32px height for digital, 0.5" for print
- Never stretch, rotate, or alter proportions
- Never place on busy backgrounds without a container
- Always use approved color variations

## Typography

### Primary Typeface: Inter
- **Headlines:** Inter Bold (700) / Inter Semibold (600)
- **Subheadlines:** Inter Medium (500)
- **Body Text:** Inter Regular (400)
- **Captions:** Inter Light (300)

### Secondary Typeface: Playfair Display
- Used sparingly for: Taglines, pull quotes, formal documents
- Weights: Regular (400), Bold (700)

### Type Scale (Web)

| Element | Size | Weight | Line Height |
|---------|------|--------|-------------|
| H1 | 48px / 3rem | Bold 700 | 1.2 |
| H2 | 36px / 2.25rem | Semibold 600 | 1.25 |
| H3 | 24px / 1.5rem | Semibold 600 | 1.3 |
| H4 | 20px / 1.25rem | Medium 500 | 1.4 |
| Body | 16px / 1rem | Regular 400 | 1.6 |
| Small | 14px / 0.875rem | Regular 400 | 1.5 |
| Caption | 12px / 0.75rem | Light 300 | 1.5 |

## Voice & Tone

### Brand Voice Attributes
- **Authoritative** - We speak with confidence and expertise
- **Approachable** - Professional but never cold or inaccessible
- **Forward-Thinking** - We emphasize innovation and progress
- **Clear** - We communicate simply and directly

### Writing Guidelines
- Use active voice
- Keep sentences concise (15-20 words average)
- Avoid jargon unless speaking to industry-specific audiences
- Lead with value and outcomes
- Use "we" for the company, "you" for the audience

---

## Imagery Style

### Photography
- Clean, modern, professional settings
- Diverse representation
- Natural lighting preferred
- Business environments: offices, meetings, cityscapes
- Avoid stock photo cliches (handshakes, pointing at screens)

### Graphics & Icons
- Line-style icons with 2px stroke weight
- Rounded corners on UI elements (8px border-radius)
- Subtle shadows for depth (no harsh drop shadows)
- Geometric patterns inspired by the logo monogram

## Subsidiary Branding Framework

Each subsidiary follows this template:

1. **Name Format:** [Subsidiary Name] -- *A Denvan Holdings Company*
2. **Logo:** Subsidiary wordmark + Denvan Holdings badge/endorsement mark
3. **Colors:** Parent Navy Blue + Sector Accent Color + White
4. **Typography:** Same Inter family (maintains consistency)
5. **Voice:** Adapted for sector audience but maintains parent brand attributes

### Example: Tech Consulting Subsidiary
- **Name:** [Name] -- A Denvan Holdings Company
- **Primary Color:** Navy Blue `#1B2A4A`
- **Accent Color:** Electric Blue `#2563EB`
- **Logo:** Custom wordmark with Denvan Holdings endorsement badge
- **Tone:** More technical, innovation-focused, still approachable
